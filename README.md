
## Main Features:

* Video player customized
* Playlist
* Preview videos
* Responsive
* It's possible crop videos
* Support with localstorage

## Contain:

- [x] React
- [x] Typescript
- [x] React Context (not Redux)
- [x] Styled components
- [x] React Router

## Build Setup

```` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run start
````
